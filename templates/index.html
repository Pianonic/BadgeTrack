Badge Generator

How to Use

  1. Enter a unique tracking tag for your project
  2. Customize the badge label, color, and style to match your project
  3. Optionally add a logo from the Simple Icons collection
  4. Click "Generate Badge" to create your dynamic badge
  5. Copy the generated markdown or HTML code
  6. Paste it into your README, website, or documentation
  7. Watch as the visit count updates automatically!
Real-time Tracking
Visit counts update instantly with each page view
Rate Limited

Customizable
